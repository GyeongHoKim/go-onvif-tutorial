This directory contains ONVIF specification PDFs downloaded from the official ONVIF website.

|------|-------------|--------|

| `ONVIF-Streaming-Spec.pdf` | Streaming specification for media transport | [onvif.org](https://www.onvif.org/specs/stream/ONVIF-Streaming-Spec.pdf) |

| `ONVIF-PTZ-Service-Spec.pdf` | PTZ (Pan-Tilt-Zoom) Service specification | [onvif.org](https://www.onvif.org/specs/srv/ptz/ONVIF-PTZ-Service-Spec.pdf) |

## Profile Specifications

Individual ONVIF Profile specifications (Profile S, T, G, A, C, M) are not available as standalone public PDF downloads. Profile requirements are defined within the Core Specification and the individual service specifications listed above.

For the complete profile feature matrix, refer to the `ONVIF-Profile-Feature-Overview.pdf` included in this directory.
